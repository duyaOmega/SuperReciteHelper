# SuperReciteHelper
一个支持多种文件和题目类型的背诵器。

## 效果简介
- 支持多种文件格式：txt、Word、PDF等。
- 支持多种题目类型：选择题、填空题、判断题、简答题等。
- 支持各种答案标记方式：红色字体、下划线、加粗、背景色等。
- 根据答题情况自动调整抽取题目的权重

## 程序说明
- quiz_app.exe为打包好的开箱即用版本，根据指示选择题库文件并确认后即可开始使用
- quiz_app.py为源代码版本，需要安装Python环境和相关依赖库，运行前请确保已安装所需库（如PyPDF、python-docx等）

## 使用方法
1. 选择题库文件：点击“选择文件”按钮，浏览并选择包含题目的文件（支持txt、Word、PDF等格式）。
2. 确认题库内容：程序会自动解析文件内容并显示题目列表，确认无误后点击“开始刷题”按钮。您可以在此处调整解析方式。
3. 进行刷题：根据提示回答题目，程序会根据答题情况自动调整抽取。目前支持鼠标点击和键盘输入两种答题方式，您可以根据个人习惯选择。
